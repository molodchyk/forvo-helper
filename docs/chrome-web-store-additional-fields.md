# Chrome Web Store Additional Fields

single_purpose: Helps Forvo contributors start recording without mouse clicks and check Ukrainian word stress through Goroh.

permission.storage: Stores extension settings and local workflow status.

permission.tabs: Opens or reuses Goroh and ChatGPT tabs and sends extension messages to supported tabs.

host_permission.forvo: Detects the current Forvo recording word and activates the record control.

host_permission.goroh: Searches Goroh and checks whether the result contains a stress mark.

host_permission.chatgpt: Inserts a fallback stress-lookup prompt into ChatGPT when enabled.

remote_code: No remote executable code is used.

privacy_policy_url: https://github.com/molodchyk/forvo-helper/blob/main/PRIVACY.md
