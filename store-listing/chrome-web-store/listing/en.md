Start pronunciation recordings without a mouse click using a keyboard command, hover delay, or optional pointer-circle gesture.

When a word is detected, open or reuse a dictionary lookup tab and check whether the result includes a Ukrainian stress mark. If no stress mark is found, optionally open a configured AI chat tab and insert a stress lookup prompt.

Settings stay in browser extension storage. No analytics, ads, tracking, telemetry, or remote hosted code are used. Browser microphone permission is still controlled by the browser and the recording site.
